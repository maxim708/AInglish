# 🤖 EnglishPal — ИИ-помощник по английскому языку

Проект для фестиваля «Область ИИзменений» (2026)

## Описание
Чат-бот ВКонтакте, который помогает ученикам 6 класса практиковать английский язык с использованием YandexGPT.

## Команда
- Участник 1: Разработка (Python, API)
- Участник 2: Промпт-инжиниринг, методика
- Участник 3: Дизайн, презентация, документация

## Технологии
- Python 3.8+
- VK Bot API
- YandexGPT API
- JSON (локальное хранилище)

## Безопасность
- Не храним ФИО и телефоны (только user_id)
- Фильтр запрещённых тем (п.44 Положения)
- Соответствие 152-ФЗ
